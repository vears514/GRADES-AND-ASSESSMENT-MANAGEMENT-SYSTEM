## Design System Overview

### Color Palette

```
Primary Colors:
- Primary Blue: #3B82F6 (main actions)
- Secondary Green: #10B981 (success/approval)
- Danger Red: #EF4444 (errors/rejection)
- Warning Orange: #F59E0B (alerts/review needed)
- Neutral Gray: #6B7280 (text/borders)

Background Colors:
- Background: #FFFFFF (light mode)
- Surface: #F9FAFB (cards/panels)
- Border: #E5E7EB (lines/dividers)

Semantic Colors:
- Success: #10B981
- Error: #EF4444
- Warning: #F59E0B
- Info: #3B82F6
```

### Typography

```
Font Family: Inter / -apple-system / BlinkMacSystemFont

Heading Sizes:
- H1: 32px (bold) - Page titles
- H2: 24px (bold) - Section titles
- H3: 20px (semi-bold) - Subsection titles
- H4: 16px (semi-bold) - Card titles

Body Text:
- Body Large: 16px (regular) - Primary content
- Body Regular: 14px (regular) - Secondary content
- Body Small: 12px (regular) - Metadata/timestamps
- Label: 12px (semi-bold) - Form labels

Line Height: 1.5 (body), 1.2 (headings)
Letter Spacing: -0.01em (headings)
```

### Spacing Scale

```
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
2xl: 48px
3xl: 64px

Padding Standards:
- Containers: 24px
- Cards: 16px
- Form fields: 12px
- Buttons: 12px 24px
```

### Border Radius

```
sm: 4px (minimal)
md: 8px (standard)
lg: 12px (cards)
xl: 16px (modals)
full: 9999px (pills/badges)
```
